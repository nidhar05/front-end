## Migrated to Vite

This project was migrated from Create React App to Vite.

Available scripts:

- `npm run dev` — Starts the Vite dev server on http://localhost:3000
- `npm run build` — Builds the production bundle with Vite
- `npm run preview` — Preview the production build locally

Install dependencies and run:

```bash
npm install
npm run dev
```
